# Legal And Media Rules

Do not:

- copy photos without permission,
- generate images that closely imitate a specific copyrighted photo,
- rewrite paid guides,
- copy someone else's GPX without checking rights,
- hide sources.

Allowed:

- write original descriptions based on multiple sources,
- quote short attributed snippets where legally appropriate,
- use OpenStreetMap data according to its license,
- use licensed stock or public domain imagery,
- generate original route maps,
- generate original AI images inspired by a place without copying a specific source image.

## AI Images

When using RouteMarket MCP image generation:

- save the returned asset key,
- record prompt, date, route, and role,
- mark it as AI-generated in `media/license_report.md`,
- use `pro` quality only for important cover images.
